=============English=============
- Install Google Chrome on your Windows 10 Machine.
- Install Python on your Windows 10 Machine.
- Open a terminal/command prompt in the directory of the project "Keno-1xBET-WS-PY", and run: pip install -r requirements.txt
- Download the Chrome WebDriver version that matches your installed Chrome version from the ChromeDriver official website.
- Extract the downloaded file using WinRAR and place the chromedriver.exe in the same directory "Keno-1xBET-WS-PY".
- In the same CMD, Run the script "keno.py" by typing: python keno.py, or run the keno.exe executable file directly.
=============Français=============
- Installez Google Chrome sur votre machine Windows 10.
- Installez Python sur votre machine Windows 10.
- Ouvrez un terminal/invite de commande dans le répertoire du projet « Keno-1xBET-WS-PY », et exécutez : pip install -r requirements.txt
- Téléchargez la version de Chrome WebDriver qui correspond à la version de Chrome que vous avez installée sur le site officiel de ChromeDriver.
- Extrayez le fichier téléchargé à l’aide de WinRAR et placez le chromedriver.exe dans le même répertoire « Keno-1xBET-WS-PY ».
- Dans le même CMD, exécutez le script « keno.py » en tapant : python keno.py, ou exécutez directement le fichier exécutable keno.exe
